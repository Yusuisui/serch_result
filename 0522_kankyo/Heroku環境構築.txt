1.Herokuアカウントの作成
https://id.heroku.com/login

2.HerokuToolbeltのインストール
 $ wget -qO- https://toolbelt.heroku.com/install-ubuntu.sh | sh

3.SSHキーの設定
 $ ssh-keygen -t rsa -C "Herokuに登録したメールアドレス"

4.Herokuログイン
 $ heroku login

5.$ heroku create アプリ名

6.$ git remote add リモート名 リポジトリアドレス
